CONDIÇÕES:

É necessário criar uma condição para indicar qual tarefa deve ser executada.
Num caixa eletrônico, o programa precisa verificar se a senha do cliente esta correta ou não.
SE estiver, um menu com opções é apresentado.
CASO CONTRÀRIO, outras operações devem ocorrer, como ler novamente a senha.
CASO o cliente selecione opções de saque, novamente vão surgir condições para serem verificadas.
Há saldo na conta?
O valor solicitado pode ser pago com as notas disponíveis no terminal?
Para definir uma condição em um programa, as linguagens de programação utilizam instruções 
próprias para essa finalidade (IF e ELSE) (SWITCH e CASE).
Os comandos para leitura de dados (ENTRADA), realização de operações (PROCESSAMENTO) e 
apresentação de mensagens (SAÍDA) são os mesmos.
O que iremos modificar agora é que alguns destes comandos serão inseridos DENTRO das 
estruturas condicionais.
Por ex: se no terminal do caixa eletrônico houver apenas notas de 10 reais e o cliente solicitar 
um valor múltiplo de 10, o pagamento pode ser realizado. 
CASO CONTRÁRIO, uma mensagem de advertência deve ser emitida.
Os comandos if...else e switch...case são os responsáveis por criar condições em JavaScript 
(e na maioria das linguagens).

IF...ELSE

Fluxograma

Exemplos da sintaxe do comando if:
// define uma condição simples
if (condição) {
    comandos
}

// define uma condição de if...else
if (condição) {
    comandos V
} else {
    comandos F
}

// define multiplas condições
if (condição 1) {
    comandos 1
} else if (condição 2) {
    comandos 2
} else {
    comandos 3
}

Quando houver apenas um comando que pertence à condição, o uso das chaves não é 
obrigatório, contudo, para facilitar a compreensão, recomenda-se utilizar { }.

OPERADORES RELACIONAIS:

==   Igual. Retorna TRUE caso os dados contenham o mesmo conteúdo.

!=   Diferente. Retorna TRUE caso os dados contenham conteúdos diferentes.

>    Maior. Pode ser utilizado para comparar números ou palavras. 

<    Menor. Também podem ser realizadas comparações de números ou palavras.

>=   Maior OU igual. Os símbolos devem estar nesta ordem (>=)

<=   Menor OU igual. Tenha cuidado com a ordem dos símbolos (<=)

===  Estritamente igual. Compara tipo e valor.

!==  Estritamente diferente. Também sobre tipo e valor.

Exemplo:
Vamos fazer um programa que faça a leitura do nome e das notas de um aluno, apresenta 
a média e uma mensagem para o aluno: "Parabéns... Você foi aprovado(a)!" ou, então,
"Ops... Você foi reprovado(a)!".
A situação de aprovado ou reprovado é definida pela média das notas, que deve ser 7.0 
ou superior para aprovação. Caso a nota seja inferior a 7.0 a mensagem indicando a 
reprovação deve ser exibida.
Também faremos nesse exemplo a aplicação de um estilo na mensagem que indica a situação.
Aplicaremos CSS (Cascading Style Sheets) para a mensagem ficar em azul se aprovado e
vermelho se reprovado.
Criar arquivo situacao-aluno.html
Criar arquivo situacaoAluno.js

Agora coloque uma terceira situação para o aluno, "Em Exame", usando ELSE IF, observe
a estrutura condicional para acrescentar essa situação.
O aluno está "Em Exame" se possuir uma média MAIOR OU IGUAL a 4 e MENOR que 7.
Se essa condição ocorre a mensagem "Atenção ${nome}. Você está em exame" deverá ser 
mostrada em LARANJA (orange).
