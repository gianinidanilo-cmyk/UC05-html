OPERADORES LÓGICOS:

No exercício anterior foi criada uma condição com base em uma comparação (media >= 7).
Há situações em que mais de uma condição deve ser analisada. Exemplos:
a) um cliente quer um carro da cor azul OU cinza;
b) o preço do carro deve ser inferior a R$ 20.000,00 E o ano MAIOR OU IGUAL a 2010;
c) o modelo do carro deve ser "fusca" E o preço MENOR QUE R$ 8.000,00.

Para definir mais de uma condição em um programa, devemos utilizar os operadores lógicos.

!   -> NOT. Indica negação. Inverte o resultado de uma comparação.

&&  -> AND. Indica conjunção. Retorna TRUE quando todas as comparações forem TRUE.

||  -> OR. Indica disjunção. Retorna TRUE se, no mínimo, uma das condições definidas for TRUE.

Mostrar tabela verdade.

Utilizando as letras (p) e (q) para representar uma proposição:
const cor = "Azul"  // (p)
const ano = 2017    // (q)

Então:
! Negação (NOT) (NÂO)
if (!cor == "Azul") { ... }
if (cor != "Azul") { ... }

&& Conjunção (AND) (E)
if (cor == "Azul" && ano == 2017) { ... }
if (cor == "Cinza" && ano < 2017) { ... }
if (ano >= 2012 && ano <= 2017) { ... }
if (cor != "Azul" && cor != "Vermelho") { ... }

|| Disjunção (OR) (OU)
if (cor == "Azul" || ano == 2017) { ... }
if (cor == "Azul" || cor == "Branco") { ... }
if ((cor == "Azul" || cor == "Branco") && ano == 2017) { ... }
if (cor == "Azul" && (ano == 2016 || ano == 2017)) { ... }

Obs: Os parênteses aqui também indicam a precedência das comparações.

OPERADOR TERNÁRIO:
É uma forma abreviada para criar instruções if ... else

const categoria = idade >= 18 ? "Adulto" : "Juvenil"

A instrução acima equivale a essa:

let categoria;
if (idade >= 18) {
   categoria = "Adulto";
} else {
   categoria = "Juvenil";
}

Ajuste agora a condição do if usada no programa peso ideal para operador ternário.
